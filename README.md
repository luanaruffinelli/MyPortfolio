# 🌐 Portfolio Personal

Proyecto desarrollado con **Astro**, **Svelte** y **Tailwind CSS** como parte de un ejercicio integrador.

---

## 🚀 Tecnologías utilizadas

* Astro
* Svelte
* Tailwind CSS
* Bun

---

## 📦 Instalación

Primero, asegurate de tener **Bun** instalado.

Luego, cloná el repositorio y ejecutá:

```bash
bun install
```

---

## ▶️ Ejecución del proyecto

Para iniciar el servidor de desarrollo:

```bash
bun dev
```

El proyecto estará disponible en:

```
http://localhost:4321
```

---

## 📁 Estructura

* `src/pages/` → páginas del sitio
* `src/components/` → componentes reutilizables
* `src/layouts/` → layout principal
* `src/styles/` → estilos globales

---

## 📝 Notas

* El proyecto utiliza **Astro con Svelte** para componentes interactivos.
* Se implementa navegación con transiciones entre páginas.
* Diseño responsive usando Tailwind CSS.
